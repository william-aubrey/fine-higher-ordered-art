**Domain:** `fhoa.org` (owned by William)

**Title treatment:** Fine^(Higher Ordered) Art — "Higher Ordered" as a mathematical exponent.

**Page hierarchy:**
- Homepage: gallery scroller → "The Charlotte Collection" → auction stats → WTA section
- Art Auction Page: rules, status, gallery scroller, bidding entry
- Winner Takes All Page: WTA explanation, current price, gallery scroller at bottom
- Image Pages: per-painting detail (images, scale diagram, narrative, video, tier buttons)
- Theme Series Pages: e.g., The Grief Series — group view + narrative
- How Bidding Works: terms, mechanism, FAQ

**Gallery:** Horizontal carousel/scroller component, not a separate page. Appears on homepage, Art Auction page, and WTA page bottom.

**Navigation flow:** Gallery Scroller → Image Page → Theme Series Page → Home

**Tier buttons:** All three visible with prices. Only lowest available tier is clickable. Others grayed out. Sequential progression enforced.

**Bidding UX split at dollar threshold (TBD by lawyer):**
- Below: self-service credit card, "Place Bid" button
- Above: lawyer-mediated, "Arrange Purchase" button, formal contract, wire transfer

**Why:** These decisions emerged from Phase 2 user journeys and are load-bearing for Phase 3 Product Design.
