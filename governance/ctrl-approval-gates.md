# Approval Gates — Fine Higher Ordered Art

*Which CXO agent approves which decision type.*
*Date: 2026-03-15*

---

| Decision Type | Approving Agent | Trigger |
|---|---|---|
| Data model / architecture changes | CAO | Any structural change to core data model, auction logic, or IDEF0 model |
| Financial commitments > $500 | CFO | Platform costs, marketing spend, art supplies bulk orders, gallery lease |
| Auction launch | CFO + Principal | Each auction event (collection listing, bid window, settlement) |
| Legal / compliance changes | CLO | Terms of service, auction terms, privacy policy, IP licensing agreements |
| Scope changes to gated features | CPO | Feature additions/removals that alter the A-0 boundary or auction mechanism |
| Revenue model changes | CRO + CFO | Pricing strategy, commission structure, tiered-bid design, payment terms |
| Technical architecture decisions | CTO | Infrastructure, framework, deployment, third-party integrations |
| Brand / public-facing messaging | CMO | Website copy, artist statement, marketing materials, social media strategy |
| Hiring / contractor engagement | CHO | Any human resource commitment (developer, photographer, gallery staff) |
| Tax posture decisions | CFO + CLO | Hobby vs. business classification, sales tax nexus, deduction strategy |
